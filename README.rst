=======
automix
=======

Automatic mix of audio clips.


------------
Installation
------------

Make sure, to have **ffmpeg** installed.

.. code-block::

    pip install -r requirements.txt


-----
Usage
-----

.. code-block::

    cd examples/zurich && automix
